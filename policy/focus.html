Privacy Policy for Focus

Effective date: April 13, 2026

Focus is an Android app designed to help users use distracting apps more intentionally. This Privacy Policy explains what information the app handles, how it is used, and the choices available to users.

Information the App Handles

Based on the current app code, Focus may handle the following information on the device:

  • Selected apps that the user chooses to monitor.
  • Focus session records such as app package name, start time, end time, session duration, written intention, and whether the goal was achieved.
  • Local app settings such as onboarding completion, reminder interval, and default threshold values.
  • Permission status related to overlay, notifications, and accessibility service setup.

How the Information Is Used

  • To detect when a monitored app is opened.
  • To show goal-entry overlays, reminder bubbles, and threshold alerts.
  • To save and display session history and dashboard statistics inside the app.
  • To remember user preferences and setup state on the device.

Accessibility and Overlay Permissions

Focus uses Android accessibility service functionality to detect when monitored apps are opened. It also uses overlay permission to display prompts and reminders on top of other apps. These permissions are used to provide the app's core mindfulness and session-tracking features.

Data Storage

Based on the repository code, Focus stores app data locally on the user's device using on-device storage technologies, including a local database and local preference storage. No remote backend service or account system was found in the repository.

Data Sharing

Focus may use service providers that help operate the app, including Firebase Analytics and Firebase Crashlytics. These providers may process technical and diagnostic information on the developer's behalf to help improve app stability, performance, and product decisions. Focus is not intended to sell personal information.

Analytics and Crash Reporting

Focus may collect analytics and crash-related information such as device type, operating system version, app version, feature usage patterns, interaction events, error logs, crash traces, and general diagnostic information. This information may be used to understand app usage, improve features, detect bugs, and fix crashes.

Analytics and crash reporting data may be collected through Firebase Analytics and Firebase Crashlytics. The exact data collected can vary depending on the Firebase features enabled in a given release and how they are configured.

Advertising ID

Focus does not use the Android Advertising ID for analytics or crash reporting, and the app is intended to be published without the AD_ID permission.

Children's Privacy

Focus is not specifically directed to children. If you believe personal information has been provided inappropriately, please contact the developer using the email below.

Data Retention and Control

Because current app data appears to be stored locally on the device, users can generally remove app data by clearing the app's storage or uninstalling the app, subject to Android system behavior.

Security

Focus uses reasonable measures intended to protect information handled by the app. No method of storage or transmission is completely secure, and absolute security cannot be guaranteed.

Changes to This Policy

This Privacy Policy may be updated from time to time. Any updated version should be published alongside the app listing or app website.
