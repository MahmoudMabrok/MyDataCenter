Privacy Policy

This privacy policy applies to the Hadith Checker app (herein referred to as "Application") for mobile devices that was created as an Open Source utility.

Information Collection and Use

The Application does not collect any personally identifiable information.

Search Queries: When you search for a Hadith or use the "Check With Hadith" menu option, the text you select or type is sent to dorar.net via their public API to retrieve search results. This data is handled according to their privacy practices.

Local Storage: The Application stores your search history and results locally on your device (using a Room database) to allow you to revisit past searches. This data remains on your device and is never uploaded to any servers by this Application.

Third Party Services

The Application uses the following third-party services which may collect information used to identify you:

  • Dorar.net API

Permissions

The Application requires the following permissions to function:

  • Internet: Used to communicate with the Dorar.net API for Hadith verification.
